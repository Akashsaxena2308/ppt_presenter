# PPT Presenter

* To get started, clone this repo
```
git clone https://github.com/chaonan99/ppt_presenter.git
cd ppt_presenter
```
* Install required packages
```
pip install -r requirements.txt
```
* Watch the video `example/test.mp4` (created by `ppt_presenter`)
